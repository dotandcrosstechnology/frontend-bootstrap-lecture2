Spacing Scale Reference

Bootstrap Spacing Scale:
  • 0 = 0rem (0px)
  • 1 = 0.25rem (4px)
  • 2 = 0.5rem (8px)
  • 3 = 1rem (16px)
  • 4 = 1.5rem (24px)
  • 5 = 3rem (48px)
  • auto = auto

Class Practice

Task: Style a Content Page using Bootstrap Utility Classes
Practice Template 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Content Page Practice</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <div class="container my-5">
        <!-- Header Section -->
        <header class="text-center mb-5">
            <h1 class="display-4 text-primary fw-bold">My Content Page</h1>
            <p class="lead text-muted">Practicing Bootstrap utility classes</p>
        </header>

        <!-- Main Content -->
        <main class="row">
            <article class="col-lg-8">
                <h2 class="fs-1 text-success mb-4">Main Article</h2>
                <p class="fs-5 lh-lg">This is the main content area where you can practice various Bootstrap utility classes.</p>
                
                <div class="bg-light p-4 rounded mb-4">
                    <h3 class="text-warning">Highlighted Section</h3>
                    <p class="text-dark">Use background and text color utilities here.</p>
                </div>

                <div class="d-flex justify-content-between align-items-center p-3 border">
                    <span class="badge bg-primary">Tag 1</span>
                    <span class="badge bg-secondary">Tag 2</span>
                    <span class="badge bg-success">Tag 3</span>
                </div>
            </article>

            <aside class="col-lg-4">
                <div class="position-sticky" style="top: 20px;">
                    <h3 class="fs-4 text-info mb-3">Sidebar</h3>
                    <div class="list-group">
                        <a href="#" class="list-group-item list-group-item-action">Link 1</a>
                        <a href="#" class="list-group-item list-group-item-action">Link 2</a>
                        <a href="#" class="list-group-item list-group-item-action">Link 3</a>
                    </div>
                </div>
            </aside>
        </main>
    </div>
</body>
</html>

Quick Reference Cheat Sheet

Typography
  • .display-1 to .display-6
  • .fs-1 to .fs-6
  • .fw-bold, .fw-normal
  • .text-center, .text-start
  • .text-uppercase, .text-lowercase
Colors
  • .text-primary, .text-secondary
  • .text-success, .text-danger
  • .bg-primary, .bg-light
  • .bg-warning, .bg-dark
Spacing
  • .m-{size} - margin
  • .p-{size} - padding
  • .mt-, .mb-, .ms-, .me-
  • .mx- (horizontal), .my- (vertical)
  • Sizes: 0, 1, 2, 3, 4, 5, auto
Display & Position
  • .d-block, .d-inline
  • .d-flex, .d-grid
  • .position-relative
  • .justify-content-center
  • .align-items-center
